GDG & DevFest

连接全球开发者的技术社区

🌐 什么是 GDG？

Google Developer Groups (GDG) 是由 Google 支持的全球开发者社区

GDG 是一个面向对 Google 技术感兴趣的开发者的社区组织，涵盖 Android、Web、Cloud、AI/ML 等多个领域。通过定期的技术分享、工作坊和社交活动，GDG 为开发者提供了学习、交流和成长的平台。

GDG 的使命

👥

建立社区

将对技术充满热情的开发者聚集在一起，创建一个互相学习、共同成长的社区环境。

📚

知识分享

通过技术演讲、工作坊和代码实验室，分享最新的技术趋势和最佳实践。

🚀

推动创新

鼓励开发者探索新技术，开发创新项目，推动技术生态系统的发展。

🎉 DevFest - 开发者节日

DevFest 是 GDG 每年举办的旗舰活动

DevFest (Developer Festival) 是全球性的年度技术盛会，由各地 GDG 社区在每年第四季度同步举办。这是一个为期一天或多天的活动，包含技术演讲、工作坊、代码实验室等多种形式。

DevFest 特色

📢 技术演讲 (Tech Talks)

行业专家和技术领袖分享最新的技术趋势、产品更新和实践经验。涵盖 Android、Web、Cloud、AI/ML、Flutter 等多个技术领域。

💻 工作坊 (Workshops)

动手实践的学习环节，参与者可以在导师的指导下完成实际项目，掌握新技术的使用方法。本次杭州站的 Chrome DevTools MCP Server 工作坊就是其中之一。

🧪 代码实验室 (Codelabs)

自定进度的编程教程，参与者可以按照详细的步骤指南，独立完成特定技术的学习和实践。

🤝 社交网络 (Networking)

与来自不同背景的开发者交流，建立联系，分享经验，寻找合作机会。这是 DevFest 最有价值的环节之一。

全球 DevFest 影响力

🌍

遍布全球

DevFest 在全球 100+ 个国家、500+ 个城市同步举办，每年吸引数十万开发者参与。

🎓

多样化内容

涵盖 Web、Mobile、Cloud、AI/ML、IoT 等多个技术方向，满足不同开发者的学习需求。

🏆

社区驱动

完全由社区志愿者组织和运营，体现了开源精神和社区力量。

🎯 2025 杭州站

杭州·阿里巴巴小镇

本次 DevFest 杭州站选择在阿里巴巴小镇举办，这里是中国互联网产业的重要聚集地。Workshop 聚焦于 Chrome DevTools MCP Server，为参与者提供了深入了解这一创新技术的机会。

活动亮点：

  • 🛠️ 深入学习 MCP (Model Context Protocol) 的使用
  • 🤖 探讨 AI 辅助编程的未来发展方向
  • 💡 与业界专家和同行开发者深入交流
  • 🎁 获得 Google 和 GDG 的官方认证和纪念品
